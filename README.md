# Vue3 自定义表单

低耦合Vue3自定义表单，以element-plus作为实现，但是对组件库不做强制绑定。
目前只支持了常见的表单项，但是可进行配置扩展。

## 支持表单项

- [x] Input 输入框
- [x] InputNumber 数字输入框
- [x] Select 选择器  
- [x] Radio 单选框
- [x] Checkbox 多选框
- [x] Date Picker 日期选择器

## 支持布局

- [x] Grid 网格
- [x] Row 行
- [x] Col 列
